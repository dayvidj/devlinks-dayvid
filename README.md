🧑‍🚀 O DevLinks é um projeto apresentado no NLW pela Rocketseat, com o objetivo de criar uma página de links personalizada, moderna e responsiva.  
O layout foi desenvolvido no Figma e a aplicação permite alternar entre tema claro e escuro, proporcionando uma experiência visual elegante e interativa.

## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

* HTML
* CSS
* JavaScript
* Figma  

## 💻 Projeto

O DevLinks é uma aplicação estilo “Link na Bio”, onde você pode reunir seus principais links (portfólio, LinkedIn, projetos, redes sociais etc.) em uma única página personalizada.

O projeto conta com:

* Alternância entre tema claro e escuro
* Design moderno com efeito glassmorphism
* Responsividade
* Ícones de redes sociais
* Interface simples e intuitiva

🔗 [Visite o projeto online](https://dayvidj.github.io/devlinks-dayvid/)
